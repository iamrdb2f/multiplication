# Tables de Multiplication

## Problème
Écrire un programme Java qui prend un entier `n` en entrée et affiche sa table de multiplication de `1` à `10`.

## Exemple d'entrée / sortie

### **Entrée : 5** 

### **Sortie :**
- 5 x 1 = 5 
- 5 x 2 = 10 
- 5 x 3 = 15 
- 5 x 4 = 20 
- 5 x 5 = 25 
- 5 x 6 = 30 
- 5 x 7 = 35 
- 5 x 8 = 40 
- 5 x 9 = 45 
- 5 x 10 = 50
